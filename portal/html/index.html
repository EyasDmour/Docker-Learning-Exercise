PG_DB Service Portal

Every container in your stack exposes a useful interface. Use the shortcuts below while you iterate on queries, data insertion, and app logic.

  • GraphQL Client React/Vite explorer for Hasura. Runs in graphql-client.
  • YouthInvestments Blank React template. Also proxied at /youthinvestments/.
  • Hasura Console Browse GraphQL schema and run queries. Container hasura.
  • pgAdmin Full PostgreSQL admin UI from pgadmin.
  • FastAPI UI Existing Python UI served by PG_UI.

Tip: keep this portal open in a pinned tab while coding. To add another service link, edit portal/html/index.html and restart the portal container.

Reverse-proxy routes

The portal also proxies each UI under predictable sub-paths:

  • /graphql-client/ → Vite dev server
  • /youthinvestments/ → YouthInvestments React app
  • /hasura/ → Hasura console
  • /pgadmin/ → pgAdmin web UI
  • /fastapi/ → Python UI
